## LangGraph
A framework that is built on top of LangChain. It helps you define graph like workflows. (nodes = steps, edges = transitions). Python/Javascript library. Basically it can help you create graphs for LangGraph Studio it's like a visual debugger it creates nodes that show step by step the workflow.
